# 易断 (YiDuan) — 易经情境诊断工具

基于易经四象定卦的情境诊断工具。通过六个问题，逐爻生成卦象，给出精准的处境分析、策略建议和自省校验。

**不是算命，是决策框架。**

## 在线体验

```bash
git clone https://github.com/greychateg/yiduan.git
cd yiduan
uv sync
uv run python -m app.main
# → http://127.0.0.1:5000
```

## 核心理念

易经的本质不是预测未来，而是**帮你看清当下**：

- 你的内在状态是什么？
- 外部环境在怎么变？
- 你处在什么阶段？
- 接下来该怎么做？

这个工具把 3000 年的易经智慧转化为一个结构化的自我诊断流程。

## 功能特性

### 四象定卦

每个问题有 4 个选项，对应易经四象：

| 四象 | 含义 | 卦爻 | 特性 |
|------|------|------|------|
| **少阳** | 稳定向好 | 阳爻 ━━━ | 正面、稳定 |
| **太阳** | 极好但在变 | 阳爻 ━━━○ | 盛极将衰，变爻 |
| **少阴** | 稳定偏弱 | 阴爻 ━ ━ | 偏弱、稳定 |
| **太阴** | 极弱但将转 | 阴爻 ━ ━× | 否极泰来，变爻 |

太阳和太阴会自动成为**变爻**——翻转后生成**变卦**，揭示局势的演变方向。

### 六问逐爻

6 个问题对应六爻的 6 个位置，从基础到结局：

1. **起点** — 你现在的基础状态
2. **内在** — 你的核心资源和能力
3. **转折** — 你面临的关键挑战
4. **外部** — 外部环境对你的态度
5. **决策** — 眼下最重要的选择
6. **走向** — 你直觉中的结果

### 六角色适配

不同身份，同样的问题有不同的解读语境：

- 📈 **投资者** — 交易决策、仓位管理、市场判断
- 🚀 **创业者** — 商业决策、融资、竞争策略
- 💼 **职场人** — 职业发展、人际关系、晋升选择
- 🔭 **观察者** — 研究分析、趋势判断、客观评估
- 📚 **求学者** — 学习方向、成长路径、能力突破
- 👑 **决策者** — 组织管理、战略规划、资源调配

### 五维分析

每卦提供 5 个维度的深度解读：

| 维度 | 回答的问题 |
|------|------------|
| **卦义核心** | 这个卦到底在说什么？ |
| **当前处境** | 我现在的真实状况是什么？ |
| **最佳策略** | 我应该怎么做？ |
| **关键风险** | 什么会让我翻车？ |
| **变化方向** | 接下来会怎么演变？ |

### 变爻与变卦

当你选择"太阳"或"太阴"时，对应的爻会**变动**：
- 太阳（阳→阴）：极盛转衰
- 太阴（阴→阳）：否极泰来

变爻翻转后形成**变卦**，展示局势的演变趋势。

### 爻位定位

生成卦象后，可以选择"你觉得自己处于哪个阶段"：

| 爻位 | 阶段 | 含义 |
|------|------|------|
| 初爻 | 潜伏期 | 还没开始，在积蓄 |
| 二爻 | 显现期 | 开始被看到 |
| 三爻 | 危险期 | 最容易犯错 |
| 四爻 | 跃迁期 | 要不要跳 |
| 五爻 | 巅峰期 | 最好的位置 |
| 上爻 | 过度期 | 该收了 |

### 自省校验

每次诊断都附带反偏差检查：
- **角色盲区** — 你这个身份最容易犯的认知错误
- **卦象反思** — 3 个针对性的深度自省问题
- **对立面思考** — 如果情况恰好相反呢？
- **上下卦自省** — 你的内在/外部判断是否准确？

## 易经基础

### 为什么是四象？

传统起卦用 49 根蓍草，经过分堆、夹取、归堆，最终得到四种结果：

- **少阳 (7)** — 概率 3/8 — 稳定的阳
- **太阳 (9)** — 概率 1/8 — 老阳，将变为阴
- **少阴 (8)** — 概率 3/8 — 稳定的阴
- **太阴 (6)** — 概率 1/8 — 老阴，将变为阳

本工具的"随机起卦"功能严格遵循这个概率分布。

### 为什么六爻？

六爻代表事物发展的六个阶段：初生 → 成长 → 转折 → 进入新阶段 → 巅峰 → 终结。下三爻（内卦）代表内在，上三爻（外卦）代表外部。这和现实决策的结构完全一致。

# 易经诊断工具 MVP 规格书

## 概述

一个基于八卦自然意象的情境诊断工具。用户描述自己的内在状态和外部环境，工具自动匹配对应的卦象，并给出判断建议和偏差提醒。

## 核心逻辑

诊断流程分两步：
1. 识别用户**内在状态** → 匹配为下卦（八卦之一）
2. 识别用户**外部环境** → 匹配为上卦（八卦之一）
3. 下卦 + 上卦 → 组合为六十四卦中的某一卦
4. 输出该卦的判断框架 + 反偏差校验提示

## 八卦意象库

每个八卦有：自然意象、核心状态描述、诊断关键词（用于匹配用户输入）。

### 1. 天（乾）☰ — 三阳
- **自然意象**：天空运转不息
- **核心状态**：全力运转、刚健、能量充沛、没有阻碍
- **内在状态关键词**：充满信心、资源充足、势不可挡、巅峰状态、什么都准备好了
- **外部环境关键词**：环境极其强势、标准极高、竞争激烈但公平、没有遮蔽

### 2. 地（坤）☷ — 三阴
- **自然意象**：大地承载万物
- **核心状态**：顺从、承载、无主动力、全面配合
- **内在状态关键词**：被动、顺其自然、没有主见、跟着走、没有独立动力
- **外部环境关键词**：环境温和无压力、大众化、没有明确方向、平坦无阻但也无助力

### 3. 雷（震）☳ — 一阳在下
- **自然意象**：雷声震动、春雷初发
- **核心状态**：突然启动、蓄积后爆发、主动出击
- **内在状态关键词**：冲劲十足、跃跃欲试、蓄势待发、想动、刚刚启动、有爆发力
- **外部环境关键词**：突发事件、市场剧变、行业洗牌、意料之外的机会或冲击

### 4. 水（坎）☵ — 一阳在中
- **自然意象**：水流穿越险阻
- **核心状态**：险阻重重、但有穿透力、持续流动
- **内在状态关键词**：被困住、四面楚歌、困难重重但还没放弃、咬牙坚持、流动中的危险
- **外部环境关键词**：市场险恶、充满陷阱、不确定性极高、黑天鹅频发、看不清底

### 5. 山（艮）☶ — 一阳在上
- **自然意象**：山岳静止不动
- **核心状态**：停止、克制、该停就停、界限明确
- **内在状态关键词**：不想动、需要休息、主动停下来、观望、等待、克制冲动
- **外部环境关键词**：环境封闭、门槛高、阻隔明显、进退不得、通道被堵

### 6. 风（巽）☴ — 一阴在下
- **自然意象**：风无处不入
- **核心状态**：柔和渗透、不硬碰、持续施加影响
- **内在状态关键词**：耐心渗透、迂回策略、不急不躁、慢慢来、柔软但坚定、顺势而为
- **外部环境关键词**：环境变化缓慢、信息逐渐传播、趋势温和、舆论/风向在转

### 7. 火（离）☲ — 一阴在中
- **自然意象**：火焰附着燃烧
- **核心状态**：光明、附丽、需要依附载体才能发挥作用
- **内在状态关键词**：看得清楚、有洞察力、但需要平台/载体、不能独立存在、依附于某个系统
- **外部环境关键词**：环境透明、信息公开、聚光灯下、高关注度、一切都被看见

### 8. 泽（兑）☱ — 一阴在上
- **自然意象**：湖泽喜悦开放
- **核心状态**：喜悦、开放、吸引、向外敞开
- **内在状态关键词**：开心、乐观、愿意沟通、开放心态、享受过程、社交活跃
- **外部环境关键词**：氛围友好、人们愿意合作、市场乐观、社交/社群活跃、大家都很high

## 已学卦象判断库

以下是已深入学习的卦象及其四问框架。工具匹配到这些卦时输出详细判断；匹配到未学卦时输出基本上下卦分析 + 提示"此卦尚未深入学习"。

### 乾卦 (1) — 天天乾 ☰☰
- **结构**：六阳，纯刚
- **力量够吗**：力量极其充沛，但过刚易折
- **时机到吗**：适合大力推进，但要注意物极必反
- **该怎么做**：自强不息，但上九"亢龙有悔"——巅峰即转折
- **最怕什么**：最怕不知道停，过度扩张

### 坤卦 (2) — 地地坤 ☷☷
- **结构**：六阴，纯顺
- **力量够吗**：没有独立的主动力量，全靠承载配合
- **时机到吗**：不是主动出击的时候，适合配合和积累
- **该怎么做**：厚德载物，顺势而为，不争先
- **最怕什么**：最怕想当主角，阴僭阳位

### 屯卦 (3) — 水雷屯 ☵☳
- **结构**：内有动力（震），外有险阻（坎）
- **力量够吗**：有初始动力但环境凶险
- **时机到吗**：万事开头难，需要耐心建立基础
- **该怎么做**：不冒进，先扎根，建立根据地
- **最怕什么**：最怕急于求成，在险地中贸然行动

### 蒙卦 (4) — 山水蒙 ☶☵
- **结构**：内陷险阻（坎），外遇阻隔（艮）
- **力量够吗**：力量不足，认知有限
- **时机到吗**：学习期，不是行动期
- **该怎么做**：虚心求教，"匪我求童蒙，童蒙求我"——主动求学
- **最怕什么**：最怕自以为是，不知道自己不知道

### 需卦 (5) — 水天需 ☵☰
- **结构**：内有刚健实力（乾），外有险阻（坎）
- **力量够吗**：力量充足，但前方有险
- **时机到吗**：时机未到，需要等待
- **该怎么做**：有实力但要等，"需于郊""需于沙""需于泥"——逐步靠近但不急入
- **最怕什么**：最怕等不住，明知有险还硬冲

### 讼卦 (6) — 天水讼 ☰☵
- **结构**：内有险（坎），外遇刚强对手（乾）
- **力量够吗**：力量不对等，对手比你强
- **时机到吗**：不适合硬拼
- **该怎么做**：中途而止，不要打到底，"讼，有孚窒惕，中吉，终凶"
- **最怕什么**：最怕赢了官司输了全局，Pyrrhic victory

### 师卦 (7) — 地水师 ☷☵
- **结构**：一阳五阴，九二独阳在地位（隐蔽的将军）
- **力量够吗**：有一个核心力量（九二），但需要授权
- **时机到吗**：需要六五（君主）授权才能动
- **该怎么做**：九二执行，六五信任，令出一门
- **最怕什么**：最怕六三冒进（中层越权），失去统一指挥
- **变量映射**：九二=根本变量，六五=活动变量，六三=突发风险变量

### 比卦 (8) — 水地比 ☵☷
- **结构**：一阳五阴，九五独阳在天位（当位得中的君主）
- **力量够吗**：君位当位得中，最理想的一阳五阴位置
- **时机到吗**：适合亲附、建立联盟
- **该怎么做**：以诚信（九五）吸引众人来亲附
- **最怕什么**：最怕来晚了，"比之无首，凶"

### 谦卦 (15) — 地山谦 ☷☶
- **结构**：一阳五阴，九三独阳在人位（山藏于地下）
- **力量够吗**：有实力但主动藏起来
- **时机到吗**：适合内敛蓄力
- **该怎么做**：有实力但保持低调，"谦谦君子，卑以自牧"
- **最怕什么**：最怕谦过头变成软弱，或者藏太久错过时机

### 豫卦 (16) — 雷地豫 ☳☷
- **结构**：一阳五阴，九四独阳在人位（不当位不中）
- **力量够吗**：有爆发力但根基空虚
- **时机到吗**：时机窗口短暂，必须速动
- **该怎么做**：借势而动（利建侯行师），但见好就收
- **最怕什么**：最怕沉迷于情绪狂热（冥豫），不知道何时退出
- **交易类比**：熊市出妖股——氛围刚爆发，人心可用，但没有基本面支撑，要速进速出
- **变量映射**：九四=根本变量（在活动层），六五=活动变量（被架空的君），初六/六三=突发风险（张扬/谄媚）
- **关键爻**：六二"介于石，不终日"= 最佳生存策略（远离震源+得中+速撤）

### 随卦 (17) — 泽雷随 ☱☳
- **结构**：三阳三阴，层层配对，均衡有序
- **力量够吗**：力量分布均衡，不是单点驱动
- **时机到吗**：适合选择和追随，"元亨利贞"四德齐备
- **该怎么做**：择善而从（孚于嘉），用极简的原则标准替代复杂的人际比较
- **最怕什么**：最怕选错追随对象（系小子失丈夫），或追随变盲从（上六拘系）
- **交易类比**：慢牛行情——到处有机会，选股比择时重要，大部分配核心+小部分切高赔率
- **变量映射**：九五=根本变量（当位得中），九四=活动变量（有力但近君须慎），初九=基层动力
- **关键爻**：九五"孚于嘉，吉"= 奥卡姆剃刀式选择；六二/六三= 机会成本的镜像描述

### 困卦 (47) — 泽水困 ☱☵
- **结构**：泽中无水，资源枯竭
- **力量够吗**：外表喜悦（兑）但内在困窘（坎）
- **时机到吗**：最低谷，但"困而不失其所亨"——即使困顿也可以亨通
- **该怎么做**：守住底线，不轻易求人，"困于石，据于蒺藜"
- **最怕什么**：最怕在困境中做出不可逆的错误决策

## 反偏差校验模块

每次诊断完成后，自动输出以下提醒：

### 下卦偏差检测（最容易出错）
提示用户：
- "你判断自己内在状态为[震/雷]。请注意：人倾向于高估自己的主动性。真的在蓄势待发，还是只是焦虑导致的冲动感？"
- "你判断自己内在状态为[坤/地]。请注意：'顺其自然'有时是'无力改变'的美化说法。你是真的在配合，还是在回避？"
- 类似的每个八卦都有一条自省提示

### 对手视角反转
- "如果你的竞争对手/交易对手来看这件事，他会怎么描述你的内在状态和外部环境？他可能把你放在哪一卦？"

### 跨域类比验证
- "找一个不同行业或不同时期的类似案例。那个案例里，内外状态的组合跟你的判断一致吗？"

## 交互流程设计（MVP）

### 输入
用户用自然语言描述当前面对的情境，包括：
1. 自己的状态（心态、资源、能力、准备度）
2. 外部环境（市场、对手、时机、趋势）

### 处理
1. 从用户描述中提取关键词，匹配八卦意象库
2. 确定下卦（内在）和上卦（外部）
3. 组合为六十四卦
4. 检索该卦的判断库

### 输出（按顺序）
1. **诊断结果**：你当前的处境对应 XX卦（上卦XX + 下卦XX）
2. **卦义简述**：一句话说明这个卦在说什么
3. **四问判断**（如果是已学卦）：力量够吗 / 时机到吗 / 该怎么做 / 最怕什么
4. **反偏差校验**：
   - 下卦自省提示
   - 上卦自省提示
   - 对手视角反转问题
   - 跨域类比验证问题
5. **关键爻提醒**（如果有）：该卦中对当前情境最相关的1-2条爻辞建议

### 注意事项
- 对于未深入学习的卦，只输出基本的上下卦分析和反偏差校验
- 所有古文均附白话翻译
- 语言风格简洁直接，用交易/决策语言，不用玄学语言
- MVP阶段先用命令行交互即可
